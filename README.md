## Installation
### As a gem

    $ gem install puppet-cic

### From github

    Tested with rvm and ruby-1.8.7

    $ git clone git://github.com/jedi4ever/puppet-cic.git
    $ cd puppet-cic
    $ gem install bundler
    $ bundle install

## Usage

### Using the Gem

    $ puppet-cic

### Using the Github version - through bundler

    $ bundle exec bin/puppet-cic
